Duplicate Checker Booleon
===
Introduction
---

A simple algorithim that returns a booleon, true or false, depending on if an array contains a duplicate interger. Written to practice writing algorithim process

Test Cases
---

|Inout |Output |
|------|-------|
|[1,1] |True |
|[1,2] |False|
|[2,2] |True |
|[1,2,3]|False|
|[2,1,2]|True|
